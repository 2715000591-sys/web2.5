# Colorful Toilet 当前稳定 UI 状态

最后整理日期：2026-05-04

这份文件记录当前已被用户认可的界面和交互基线。它不是发布流水。改 UI 前先看这里，避免顺手改坏稳定体验。

## 1. 当前结论

当前稳定版本：

- `BUILD_ID=2026-05-04-1159`
- 扩展版本：`0.1.76`
- App / Extension 版本：`1.0.76 (77)`
- Worker Version ID：`febe67c9-e44a-4f28-a563-f71c0f25e4f2`

当前 UI 主体已被用户认可：

- 官网首页形态不要重做。
- 控制台形态不要重做。
- 右栏关闭能力是稳定资产。
- 底部整理卡片要保持数据真实性，不要把旧历史说成“你刚标记”。

## 2. 右栏关闭稳定行为

当前右栏这七类模块可以手动关闭：

- `订阅 Premium`
- `X 上的直播`
- `你可能会喜欢`
- `相关用户`
- `有什么新鲜事`
- `推荐关注`
- `服务条款 / 隐私政策 / Cookie 政策 / 辅助功能 / 广告信息 / 更多`

必须守住：

- 每块右上角有灰色小叉。
- 点叉只关闭自己。
- 关闭后下面模块自然顶上来。
- 不能留下白色空框。
- 不能让主时间线、左栏、顶部导航消失。
- X 局部重渲染后，插件要能补回小叉。
- 真正刷新页面后模块会重新出现，用户可以再次关闭。

当前口径：右栏关闭是“当前页面有效”，不是“跨刷新永久记忆”。

## 3. 用户开关

右栏关闭能力不是强制塞给用户：

- Safari 插件弹窗里有开关。
- 官网公开页里也有对应偏好。
- 用户登录后，偏好尽量在官网和插件之间保持一致。
- 用户主动关闭时，不要继续注入右栏灰叉。
- 用户开关优先级高于默认展示能力。

## 4. 控制台稳定口径

控制台累计成果入口应集中在 `你的累计成果` 方格里，不要再单独堆一块来源入口。

当前主要方格：

- `累计跳过无用内容`
- `后台直接下沉`
- `后台学习库下沉`
- `你手动冲走`
- `跳过官方广告`

用户可见文案统一叫“后台”，不要强迫用户区分 AI 后台、数据库后台、规则后台。

旧来源归并口径：

- 旧 AI 记忆、数据库历史命中、公共数据库规则、账号黑名单、本地历史规则，用户界面优先归入 `后台学习库下沉`。
- 确实由用户新点击的 `manual_hide` 才能说 `你手动冲走`。
- 除非有新的 `manual_hide` 事件，不能写成“你刚标记”。
- 恢复误判后，当前列表、累计明细、最近记录和底部卡片都要移除对应当前屏蔽记录；原始历史保留在数据库里追溯。

## 5. 底部整理卡片

底部卡片必须尊重当前详情页已下沉缓存：

- X 展开底部或页面重排时，可能临时把隐藏回复从页面里卸掉。
- 这时不能把按钮改成“无下沉”。
- 不能清空恢复列表。
- 不能把按钮置灰。
- 只有用户恢复、后台明确放过，或切到其他页面 / 详情页时，才能清掉对应缓存。

等待后台判断的内容显示为“后台复审中”。它不是已下沉结果，也不能算进本机自动下沉。

## 6. 设备显示口径

控制台 `已接入设备` 不要逐条展示底层 `device_id`。

- 主界面突出当前这台设备和最近活跃时间。
- 历史连接标识折叠说明。
- `device_dev_*`、`device_eval_*`、`device_test_*` 不当作真实用户设备展示。
- 这是展示口径，不删除历史数据，不改数据库结构。

## 7. 不要轻易改的地方

没有用户明确要求时，不要：

- 改右栏叉子的视觉位置。
- 把“刷新恢复”改成永久记忆。
- 用粗暴 `display:none` 直接动大外层宿主。
- 删除安全壳层判断。
- 借修 bug 重设计主页或控制台。
- 顺手改筛选、AI、头像证据、数据库学习主链路。

## 8. UI 改动后怎么验证

基础检查：

```bash
node --check extension/content/content.js
node --check site/app.js
```

影响 Safari 扩展时：

```bash
npm run safari:build
codesign --verify --deep --strict --verbose=2 /Applications/web2.5.app
npm run safari:verify-live
```

影响官网或控制台公网时：

```bash
npm run cloud:check
npm run cloud:deploy
```

真实页面至少确认：

- `document.documentElement.dataset.web25Build` 是当前 `BUILD_ID`
- 详情页有可见 `冲走` 按钮
- 右栏有可见关闭按钮
- 底部查看列表展开、收起、等待后仍稳定
