# 色情引流基础提示词包

最后整理日期：2026-05-01

这个目录是回复区 AI 审核的公共基础提示词包。它只处理大多数用户都不想看到的公共垃圾：

- 色情/约见引流
- 联系方式导流
- 主页/置顶/简介诱导
- 空洞低信息诱饵
- 诈骗、木马、安装包、资源包风险

它不处理个人喜好。普通色情内容、成人话题、色情讨论、性教育、平台治理讨论、玩笑、观点和上下文内的正常表达，默认应该保留。

## 文件

- `prompt.md`：给 AI 的基础审核口径。
- `samples.json`：用于人工回归和以后扩充的样本集合。

## 当前接入方式

这套材料已经接入云端回复 AI 审核。`cloudflare/src/reply-ai-prompt-pack.generated.js` 由本目录自动生成，`cloudflare/src/index.js` 会读取这个生成文件，把教材和样本放进 AI 老师的判断提示词里。

改本目录后运行：

```bash
npm run prompt-pack:sync
```

`npm run doctor` 和 `npm run cloud:check` 会检查生成文件是否和本目录一致，避免教材改了但程序没读到。

## 更新规则

- 新增样本时，必须写清楚 `expectedAction` 是 `hide` 还是 `allow`。
- `manual_hide` 可以作为垃圾候选样本。
- `manual_allow` 只表示“这条不要按垃圾处理 / 不要升级公共规则”，不能当成另一种内容的正样本。
- 不要把单个用户反复冲走直接升级成公共规则；公共规则需要多贡献者或开发者确认。
- 不要把完整 API Key、用户隐私、cookie、session 或未脱敏个人信息写进这个目录。
