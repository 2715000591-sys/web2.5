# Colorful Toilet 网页控制台

这个网页不是替代 Safari 插件，而是给 `Colorful Toilet` 做一个公开前台。

## 这版先解决什么

- 让任何设备都能看到产品介绍和下载入口
- 把“被整理掉的内容”从 X 页面内移到网页控制台里审查
- 给以后接 API、接跨设备同步、接账号系统留位置

## 为什么现在就要做网页

- Safari 扩展本身更适合做过滤，不适合承载大量“审查记录”界面
- 网页适合承接：
  - 下载扩展
  - 查看最近整理记录
  - 查看高频风险词语
  - 未来配置 API
  - 未来管理账号与同步

## 当前公网状态

网页控制台已经跑在 Cloudflare 公网上，不再是“还没部署”的计划稿。

当前入口：

- 官网：`https://colorful-toilet.colorful-toilet.workers.dev/`
- 控制台：`https://colorful-toilet.colorful-toilet.workers.dev/console/`

后续如果继续改官网、控制台或下载包，验证后要发布到 Cloudflare，不能只停在本地预览。

## 后续最适合的架构

1. 扩展负责本地过滤与轻量记忆
2. 网页负责控制台、下载、审查、同步密钥、API 配置
3. 后端负责：
   - 风险账号数据库
   - 共享黑名单
   - AI 调用队列
   - 跨设备同步

## 当前最小同步方案

现在先不做正式登录，先做一个轻量版本：

- 扩展自动生成 `同步密钥`
- 扩展把你手动 `冲走 / 恢复此条` 的事件写进后台
- 网页控制台通过 `syncKey` 读取属于你的数据

这样先解决两件事：

- 不同人的数据不会混在一起
- 任何设备都可以通过同一把同步密钥看到同一份数据

## 建议的数据视图

- 今日整理数量
- 本周手动标记数量
- 高频话术列表
- 最近被整理的回复
- 系统自动整理 vs 你手动处理
- 未来的 API 使用状态
