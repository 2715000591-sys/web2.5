Mac / Safari

Safari 用户点这里

下载后打开安装包，按提示开启 Safari 扩展。

下载 Safari 完整版 备用压缩包
  1. 打开下载好的安装包。
  2. 把 Colorful Toilet.app 拖到 Applications。
  3. 打开 app，再到 Safari 设置里开启扩展。
  4. 打开 x.com，用自己的邮箱登录控制台。

Safari 不允许网站偷偷自动安装扩展，所以需要你手动开启一次。

Chrome / Edge

Chrome 或 Edge 点这里

这两个浏览器用同一个插件包。

下载 Chrome / Edge 包
  1. 下载并解压。
  2. 打开浏览器的扩展管理页。
  3. 打开开发者模式，选择刚解压的插件文件夹。
  4. 打开 x.com 开始使用。

以后发布到商店后，这一步还能继续变简单。
